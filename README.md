# IPL_Analysis_Dashboard_PBI

# 🏏 IPL Analysis Dashboard (2008–2025) | Power BI

## 📊 Project Overview

This project showcases an **interactive IPL (Indian Premier League) Analysis Dashboard** built using **Power BI**, covering seasons from **2008 to 2025**.

The dashboard is designed to provide comprehensive insights into **team performance, player achievements, and match statistics** through visually rich and interactive reports. It enables users to explore season-wise trends and identify top-performing players and teams.

---

## 🎯 Business Objective

* Deliver a centralized view of IPL statistics
* Track player performance (batting & bowling)
* Analyze team standings and match outcomes
* Enable quick decision-making using interactive visuals

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in **Power BI Desktop**
3. Select a season from the slicer (top-right)
4. Explore:

   * Player stats
   * Team standings
   * Match insights
